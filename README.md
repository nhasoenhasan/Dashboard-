## Run Setup

 1. Clone repository
    `$ git clone https://github.com/nhasoenhasan/Dashboard-.git`

 2. Run Website


    - copy folder to yout root server folder

    - open web browser with url localhost



## Stacks and Dependencies

 - Boostrap 4.4 
 - Fontawesome 5.12.0
 - Chart Js
